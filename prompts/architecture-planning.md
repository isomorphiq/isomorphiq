# Architecture & Planning Prompts

## System Design Prompt
```
Design a comprehensive architecture for a [FEATURE/SYSTEM]. Consider:
- Core components and their responsibilities
- Data flow and communication patterns
- Scalability and performance requirements
- Security considerations
- Technology stack recommendations
- Deployment strategy

Provide detailed diagrams, component specifications, and implementation roadmap.
```

## API Design Prompt
```
Design a RESTful API for [FEATURE]. Include:
- Resource endpoints and HTTP methods
- Request/response schemas
- Authentication and authorization
- Error handling patterns
- Rate limiting and pagination
- API versioning strategy

Generate OpenAPI/Swagger specification.
```

## Database Schema Prompt
```
Design a database schema for [APPLICATION]. Consider:
- Entity relationships and constraints
- Indexing strategy for performance
- Data migration plan
- Backup and recovery procedures
- Query optimization patterns

Provide SQL DDL statements and ER diagrams.
```
